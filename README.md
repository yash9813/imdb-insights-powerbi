## Project Overview
This project presents an interactive Power BI dashboard built to analyze IMDb movies and TV shows data.  
The dashboard provides insights into ratings, genres, trends, and overall performance of movies and shows.

## About This Repository
The original Power BI (.pbix) file was accidentally deleted from my system.  
However, a complete screen recording of the working dashboard is available as a demonstration.

## Objectives of the Project
- Analyze movie and TV show ratings
- Identify most popular genres
- Understand trends over years
- Compare ratings with votes
- Present insights through visual storytelling

## Tools Used
- Power BI  
- Data Visualization  
- Analytical Thinking  

## Key Insights Generated
- Distribution of IMDb ratings  
- Top-rated movies and shows  
- Genre-wise performance  
- Year-wise trends  
- Relationship between votes and ratings
